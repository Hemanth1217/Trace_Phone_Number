# Trace_Phone_Number
To find the details of the phone number like which country , sim card providers , lattitude and longitude , using the python modules such as phonenumbers,OpenCage Geocoding,folium.


1.Phonenumbers Python Library: https://pypi.org/project/phonenumbers/

2.OpenCage Geocoding Module for Python: https://pypi.org/project/opencage/

3.OpenCage Geocoding API: https://opencagedata.com/

4.Folium: https://pypi.org/project/folium/

5.PyCharm (Code Editor): https://www.jetbrains.com/pycharm/
